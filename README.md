# Cloud Security and Governance Platform

Enterprise-style cloud security and governance platform built on Microsoft Azure.

## Core Technologies

- Azure Functions
- ASP.NET Core Minimal APIs
- Terraform
- PostgreSQL
- Azure Monitor
- Application Insights
- Managed Identities
- Microsoft Entra ID
- GitHub Actions
- Docker

## Project Goals

- Azure resource discovery
- Cloud security posture scanning
- Governance compliance evaluation
- Findings management
- Alerting and remediation
- Enterprise-grade observability
- Production-style CI/CD

## Repository Structure

- `infrastructure/` → Terraform IaC
- `src/functions/` → Azure Functions
- `src/api/` → ASP.NET APIs
- `src/shared/` → Shared libraries
- `src/workers/` → Background workers
